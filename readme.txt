Welcome to the Artec Naqsh Project Repository!

ABOUT ARTEC NAQSH:
------------------
Artec Naqsh is a leading architecture and design firm based in Tashkent, Uzbekistan, specializing in creating unique and culturally enriched environments. Our projects span across residential, commercial, and public sectors, integrating modern design with traditional Uzbek aesthetics.

This repository contains resources, documentation, and project files related to our architectural designs, interior projects, and collaborations.

DIRECTORY STRUCTURE:
--------------------
/docs       - Project documentation and manuals.
/designs    - Architectural and interior design files.
/assets     - Images, logos, and other media used in projects.
/scripts    - Automation scripts and utilities.

USAGE:
------
The files and resources contained in this repository are intended for collaboration and reference for Artec Naqsh projects. Please ensure you have the necessary permissions to use or modify the content.
